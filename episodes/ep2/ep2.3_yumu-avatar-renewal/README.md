# ep2.3 ゆむのアバターを全部作り直した話 — Bing Image Creator でキャラ絵を作るときのプロンプトの考え方

動画「ゆむのアバターを全部作り直した話」で実際に使ったプロンプトのメモです。

技術的な実装はなく、Bing Image Creator への指示文の試行錯誤の記録です。

---

## プロンプトの基本方針

- **感情・ポーズは具体的な言葉で指定する**
- **テイスト・絵柄・サイズは「変えないで」と一言添えて AI に委ねる**

```
この画像のテイストと絵柄とサイズは全く変えずに、〇〇している感じの画像を制作してください。
背景は白にしてください。
```

---

## 各表情のプロンプト

### Stand（一発生成）

```
この画像のテイストと絵柄は全く変えずに、全身絵を制作してください。
全身絵は画面内に収まるようにして、背景は白にしてください。
```

### Chibi

Stand 画像を手動でトリミングして作成。

### Joy（思ってたのと違う画像ができたが、Joy っぽかったので採用）

```
この画像のテイストと絵柄とサイズは全く変えずに、リラックスしている感じの画像を制作してください。
背景は白にしてください。
```

### Thinking（一発生成）

```
この画像のテイストと絵柄とサイズは全く変えずに、考え込んでいる感じの画像を制作してください。
背景は白にしてください。
```

### Surprise（少し試行錯誤）

**失敗** — テイストが崩れた。本当はこれを使いたかったが見送り

```
この画像のテイストと絵柄とサイズは全く変えずに、驚いている感じの画像を制作してください。
背景は白にしてください。
```

**成功** — なぜ崩れなくなったかわからないが、少し大げさな表現でこれを採用

```
この画像のテイストと絵柄とサイズは全く変えずに、すごく驚いている感じの画像を制作してください。
背景は白にしてください。
```

### Angry（試行錯誤）

**失敗1** — 怒りすぎ

```
この画像のテイストと絵柄とサイズは全く変えずに、少し腹が立ってむっとしている感じの画像を制作してください。
背景は白にしてください。
```

**失敗2** — 別人になった（動画内の失敗例）

```
この画像のテイストと絵柄とサイズは全く変えずに、少しむっとしている感じの画像を制作してください。
背景は白にしてください。
```

「むっとして」が怒りとして伝わらず、別の感情や身体的な特徴として解釈された。

**失敗3** — 怒っているが、のっぺりした感じになった

```
この画像のテイストと絵柄とサイズは全く変えずに、少しムカついている感じの画像を制作してください。
背景は白にしてください。
```

**採用** — 他の表情とテイストが少し違うが許容範囲。あまり使わない感情なので妥協

```
この画像のテイストと絵柄とサイズは全く変えずに、少しイラだっている感じの画像を制作してください。
背景は白にしてください。
```

### Sad（思ってたのと違う画像ができたが、Sad っぽかったので採用）

```
この画像のテイストと絵柄とサイズは全く変えずに、少し悩んでいる感じの画像を制作してください。
```

### Troubled（少し試行錯誤）

**失敗** — 悲壮感が強すぎた

```
この画像のテイストと絵柄とサイズは全く変えずに、静かに悲しんでいる感じの画像を制作してください。
背景は白にしてください。
```

**成功** — ゆむらしい「困っている感」が出た

```
この画像のテイストと絵柄とサイズは全く変えずに、少し困っている感じの画像を制作してください。
背景は白にしてください。
```
